## 🔍 Overview

This repository contains practical implementations of core machine learning algorithms, from basic data exploration to advanced regression techniques. Each project includes detailed Jupyter notebooks with explanations, visualizations, and real-world applications.

## 🚀 Projects

### 1. Linear Regression
- **Energy Efficiency Analysis**: Predicting heating and cooling loads based on building characteristics
  - Features: 8 input features including relative compactness, surface area, wall area, etc.
  - Implementation includes standardization, cross-validation, and residual analysis
  - Files: `Energy efficiency.ipynb`, `energy_efficiency.py`, `implementation.ipynb`

### 2. Logistic Regression
- Binary classification implementation
- Files: `implementation.ipynb`

### 3. Ridge and Lasso Regression
- Regularization techniques for linear regression
- Implementation of Ridge regression (L2 regularization)
- Files: `implementation Ridge.ipynb`

### 4. Understanding Data
- Comprehensive data exploration and analysis
- Key questions covered:
  - How big is the data?
  - What does the data look like?
  - What are the data types?
  - Are there missing values?
- Files: `1.ipynb`, `train.csv`

### 5. Toy Project
- Experimental machine learning project
- Files: `main.ipynb`
